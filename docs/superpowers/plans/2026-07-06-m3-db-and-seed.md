# M3 DB & Seed Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Persist the M3 domain ‚Äî add the community/economy tables (`projects`, `contributions`, `market_listings`, `chat_messages`) and the character skill columns (`learned_skills`, `equipped_skills`) to the Drizzle schema, migrate, restore the API to green by emitting `skills[]` in the character DTO, and seed the belfry project `r1.belfry`.

**Architecture:** Drizzle schema in `apps/api/src/db/schema.ts` mirrors `docs/DATA-MODEL.md`; migrations are generated by `drizzle-kit generate` and applied by `drizzle-kit migrate`. Character skill state lives in two JSONB columns and is projected into the API `skills[]` contract at read time. Seeds are idempotent upserts.

**Tech Stack:** Drizzle ORM (postgres), drizzle-kit, PostgreSQL 16 (running via `docker compose up -d db`), Vitest, tsx.

export const marketListings = pgTable(
  'market_listings',
  {
    id: uuid('id').defaultRandom().primaryKey(),
    sellerId: uuid('seller_id')
      .notNull()
      .references(() => characters.id),
    itemId: varchar('item_id', { length: 40 })
      .notNull()
      .references(() => items.id),
    qty: integer('qty').notNull(),
    unitPrice: integer('unit_price').notNull(),
    createdAt: timestamp('created_at', { withTimezone: true }).notNull().defaultNow(),
  },
  (t) => [
    index('market_item_idx').on(t.itemId, t.unitPrice),
    check('listing_positive', sql`${t.qty} > 0 AND ${t.unitPrice} > 0`),
  ],
);

// ‚îÄ‚îÄ‚îÄ Social: chat (M3 ‚Äî global + region channels) ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ
export const chatMessages = pgTable(
  'chat_messages',
  {
    id: uuid('id').defaultRandom().primaryKey(),
    channel: varchar('channel', { length: 48 }).notNull(), // global | region:1

      .notNull()
      .references(() => characters.id),
    body: varchar('body', { length: 500 }).notNull(),
