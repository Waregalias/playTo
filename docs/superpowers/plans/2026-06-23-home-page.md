# Home Page — Cozy Kingdom Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implémenter la home page de Cozy Kingdom (hero, features, quote) avec une modale de connexion par jeton qui navigue vers `/game`.

**Architecture:** Quatre composants standalone — `HomeComponent` (page), `LoginModalComponent` (modale), `GameComponent` (placeholder destination). `HomeComponent` contrôle la visibilité de la modale via un signal `isModalOpen`. Routing Angular 22 avec lazy-loading sur les deux routes. Tailwind v4 avec tokens personnalisés via `@theme` en CSS.

**Tech Stack:** Angular 22 standalone, Tailwind CSS v4 (`ng add tailwindcss`), Google Fonts Cinzel + Lora, Vitest 4 (globals configurés via `tsconfig.spec.json`), TypeScript 6.

## Global Constraints

- Aucun commentaire mentionnant "claude" ou "ai" dans les fichiers
- Pas d'images réelles — placeholders colorés Tailwind/inline style
- Aucune librairie UI externe hormis Tailwind CSS
- Standalone components Angular 22 partout — pas de `NgModule`
- Utiliser les nouvelles syntaxes : `@if`, `@for`, `@defer`, `signal()`, `output()`
- Les globals Vitest (`describe`, `it`, `expect`, `vi`, `beforeEach`) sont disponibles sans import grâce à `tsconfig.spec.json` — ne pas les importer

```ts
import { Component, output, signal } from '@angular/core';
import { Router } from '@angular/router';

@Component({
  selector: 'app-login-modal',
  templateUrl: './login-modal.html',
})
export class LoginModalComponent {
  closed = output<void>();
  token = signal('');

  constructor(private router: Router) {}

  close(): void {
    this.token.set('');
    this.closed.emit();
  }

  submit(): void {
    if (this.token()) {
      this.router.navigate(['/game']);
    }
  }

  onBackdropClick(event: MouseEvent): void {
    if (event.target === event.currentTarget) {
      this.close();
    }
  }
}
```
